Real-Time Game Design of Pac-Man Using Fuzzy Logic

Authors: Adnan Shaout, William King

This project presents a fuzzy logic–based AI implementation in a real-time Pac-Man game. Unlike classic rule-based ghost behavior, fuzzy logic is used to make ghost agents respond more naturally to Pac-Man's position, distance, and environment.

Design Approach

  • Ghosts are BDI-style agents (Belief, Desire, Intention).
  • Fuzzy inference engines govern movement patterns and pursuit strategies.
  • Rules use degrees of danger, direction, and distance rather than fixed values.

Example Rules

If DistanceToPacman is Near AND Direction is Front THEN Chase
If DistanceToPacman is Far AND Direction is Back THEN Patrol

Conclusion

Integrating fuzzy logic into Pac-Man creates a more complex, immersive game environment, demonstrating how even simple AI systems benefit from soft decision-making methods.

Fuzzy Control for Maze-Solving Robot

Author: Cheng Guoyi

This project implements a fuzzy controller to navigate a two-wheeled robot through a maze. The goal is to replace traditional PID control with a flexible fuzzy system that adapts to real-world uncertainty.

Robot Setup

  • Robot has left and right wheel motors.
  • Distance sensors detect proximity to walls.
  • Control system adjusts wheel speed to avoid obstacles and stay centered.

Fuzzy Rules

If LeftSensor is Close AND RightSensor is Far THEN Turn Right
If BothSensors are Close THEN Move Backward

Results

The robot successfully navigated mazes with varying widths. The fuzzy system smoothly transitioned through corners and avoided sharp turns better than hard-coded logic.

Conclusion

Fuzzy logic allows better handling of uncertainties and makes robotic systems more adaptable in unstructured environments like mazes.

FuzzySense Runner – Interactive AI with Fuzzy Logic

Source: ResearchGate Study

The FuzzySense Runner is inspired by robotics projects that use fuzzy controllers to enhance decision-making in interactive scenarios. The study demonstrates fuzzy AI applied to a robot that reacts based on power level, distance to enemies, and sensory data.

AI Logic

  • Battery levels influence speed and risk-taking.
  • Proximity sensors influence dodging vs running.
  • Rules ensure the robot adapts rather than follows hard commands.

Example Rule

If Battery is Low AND ObstacleDistance is Close THEN Stop and Recharge

Use Case

Robots controlled by fuzzy systems are more responsive and less brittle. Games using such models can simulate more realistic, lifelike AI.

Conclusion

FuzzySense Runner showcases how fuzzy logic makes decision-making smooth and continuous, ideal for games where player unpredictability is high.

Fuzzy Logic for Dynamic Robot Navigation

Author: Dr. Amelia Hawke

This article delves into the dynamic challenges faced by mobile robots navigating unpredictable environments, such as busy city streets. By applying fuzzy logic, robots can make real-time decisions that allow them to better adapt to environmental changes such as unexpected obstacles, changes in traffic, or sudden shifts in weather conditions.

Key Features of the System

  • Adaptive decision-making based on sensory feedback.
  • Fuzzy logic enables continuous re-evaluation of obstacles.
  • Real-time adjustments made for speed and navigation paths.

Results

The fuzzy control system outperformed conventional methods in real-world environments, where sensors often provide imprecise or noisy data. The robot was able to maintain smoother, more efficient paths even in uncertain and rapidly changing conditions.

Conclusion

Fuzzy logic enhances robot navigation by adding an adaptive layer that mimics human decision-making, improving robot autonomy and safety in dynamic environments.

Real-Time Fuzzy Logic for Autonomous Vehicles

Author: Dr. Mark Thompson

This research explores how fuzzy logic is applied to autonomous vehicles for real-time decision-making. Fuzzy controllers help manage uncertain road conditions, unexpected obstacles, and interactions with other vehicles. The system integrates sensor data to determine vehicle speed, braking force, and steering adjustments.

System Features

  • Real-time sensor data interpretation (e.g., LIDAR, cameras).
  • Dynamic decision-making under various driving conditions.
  • Safety systems that adapt to road environments like traffic congestion, accidents, or weather changes.

Results

The fuzzy logic system was able to successfully navigate complex driving scenarios, such as sudden lane changes or unexpected obstacles, without requiring pre-programmed responses.

Conclusion

Fuzzy logic contributes to the development of safer and more flexible autonomous driving systems by providing real-time adaptability, crucial for unpredictable driving environments.
